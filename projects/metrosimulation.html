Problem description
For order to be maintained on the metro, the is an upper limit to the number of people allowed into the station. Each station runs concurrently with every other station, and the lack of communication between stations means that a simple atomic read/write solution cannot work. There is, however, a global overseer in the network, who communicates periodically with the stations.
Solution
To acquire an accurate count, the overseer has the ability to aquire a synchronized state with each station, and passes instructions on whether to allow the flow in of passengers or to only allow passengers off.
