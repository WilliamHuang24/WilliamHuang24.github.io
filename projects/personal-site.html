Gravity bubble chart
For the main page, I wanted to have something interactive rather than the usual about me section. So instead, I created a interactive bubble chart. Each bubble in the chart is representative of a project, with size proportional to the number of commits the project has.
Forces and interaction
The forces acting on the bubbles are done through d3's force library. Interaction is done with custom JavaScript, editing the inline style on mouse events such as showing which elements are highlighted. A custom drag event is written to handle the moving of bubbles and passed into the d3 simulation.
Grouping
Grouping was inspired by LayerCake's component using similar methods and svelte reactivity. A boolean state was bound the checkbox, and conditional force is applied to the simulation to seperate bubbles into different categories.
